# Chapter 10: 분포 변화 감지 (Distribution Shifts)

이 디렉터리는 에이전트 운영 환경에서 데이터 분포의 변화(Drift)를 감지하고 대응하는 기법을 다룹니다.

## 파일 목록

*   `distribution_shifts.py`: 입력 데이터의 통계적 변화(KS 검정), 확률 분포 변화(KL 발산), 범주형 데이터 변화(PSI), 임베딩 기반 의미론적 변화 등을 감지하는 다양한 함수와 예제를 포함합니다.

## 실행 방법

```bash
python distribution_shifts.py
```
